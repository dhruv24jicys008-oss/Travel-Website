
# TripGlide – Travel Website

TripGlide is a modern and responsive travel website designed to help users explore destinations, view travel packages, and book unforgettable trips with ease.

# Features

1. Attractive homepage with hero section

2. Easy navigation between multiple pages

3. Highlighted features and benefits

4. Traveler testimonials section

5. Call-to-action for exploring destinations

6. Responsive layout for all devices

# Technologies Used

1. HTML5 – Website structure

2. CSS3 – Styling and layout

3. JavaScript – Interactivity and slider functionality
